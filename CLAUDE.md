# winning-with-claude

Open-source blueprint for structuring Claude Code context in any project.

## Files

- `winning-with-claude.md` — full guide: layered architecture, templates, case study
- `SKILL.md` — installable Claude Code skill (`claude-project-setup`)

## Usage

Install the skill by placing these files at:

```
.claude/skills/claude-project-setup/
├── SKILL.md
└── references/
    └── winning-with-claude.md
```
